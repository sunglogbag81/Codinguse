Chapter 9. 자바스크립트의 영역 확장

Node.js 백엔드 (Full Course)

브라우저에 갇혀 있던 자바스크립트를 컴퓨터(서버)로 꺼내준 혁명적인 기술입니다. 새로운 언어를 배울 필요 없이, 프론트엔드와 동일한 JS로 서버를 구축해 보세요.

1. Node.js란?

Node.js는 언어가 아닙니다. 자바스크립트가 브라우저 밖(내 컴퓨터, 서버)에서도 돌아갈 수 있게 해주는 '실행 환경(런타임)'입니다.

첫 번째 Node 프로그램 실행하기

메모장에 JS 코드를 적고, 터미널에서 node 파일명을 입력하면 브라우저 없이도 코드가 실행됩니다.

// app.js 파일 const name = "개발자"; const age = 20; console.log("Node.js의 세계에 오신 것을 환영합니다!"); console.log(name + "님은 " + age + "살 이군요.");
> Terminal - node app.js
Node.js의 세계에 오신 것을 환영합니다!
개발자님은 20살 이군요.

2. 웹 서버 띄우기 (Express.js)

Node.js로 서버를 밑바닥부터 짜면 코드가 너무 길어집니다. 그래서 전 세계의 99%는 Express.js라는 마법의 도구(프레임워크)를 사용해 단 5줄 만에 서버를 켭니다.

// 1. express 모듈(남이 만든 코드)을 불러옵니다. const express = require('express'); // 2. app 이라는 이름으로 서버 객체를 만듭니다. const app = express(); // 3. 서버를 3000번 포트에서 켭니다. app.listen(3000, () => { console.log("서버가 3000번 포트에서 돌아가는 중! 🚀"); });
> Terminal - node server.js
서버가 3000번 포트에서 돌아가는 중! 🚀
💡 포트(Port)란? 서버라는 커다란 아파트에 들어가는 '호수'입니다. 데이터베이스는 3306호, 리액트는 3000호, 노드 서버는 주로 8080호나 3000호를 사용합니다.

3. 라우팅 (Routing)과 요청/응답

사용자가 /login 주소로 오면 로그인 화면을, /main으로 오면 메인 화면을 보여주는 등 길(Route)을 안내해 주는 작업을 라우팅이라고 합니다.

req (요청) 와 res (응답)

클라이언트(브라우저)가 보낸 편지는 req(request)에 담기고, 서버가 돌려줄 편지는 res(response)를 통해 보냅니다.

const express = require('express'); const app = express(); // 누군가 기본 주소('/')로 접속(GET)하면? app.get('/', (req, res) => { res.send("여기는 메인 페이지입니다."); }); // 누군가 '/hello' 주소로 접속하면? app.get('/hello', (req, res) => { res.send("안녕하세요, 반가워요!"); }); app.listen(3000);
> Browser - http://localhost:3000/hello
안녕하세요, 반가워요!

4. JSON 데이터 보내기 (API 만들기) ★중요

요즘 백엔드 서버는 HTML 화면을 직접 그려서 보내주지 않습니다. 대신 리액트나 스마트폰 앱이 이해할 수 있는 'JSON'이라는 데이터 덩어리만 툭 던져주는데, 이를 API(에이피아이)라고 부릅니다.

app.get('/api/users', (req, res) => { // 데이터베이스에서 가져왔다고 치는 가짜 데이터 const userList = [ { id: 1, name: "홍길동", role: "admin" }, { id: 2, name: "김철수", role: "user" } ]; // res.send 대신 res.json()을 사용하면 // 객체/배열을 예쁜 JSON 형태로 변환해서 보내줍니다. res.json(userList); });
> API Response (GET /api/users)
[
  {
    "id": 1,
    "name": "홍길동",
    "role": "admin"
  },
  {
    "id": 2,
    "name": "김철수",
    "role": "user"
  }
]
💡 프론트엔드와 백엔드의 분리 이런 식으로 Node.js 백엔드가 res.json()으로 데이터만 쏴주면, 전에 배웠던 React가 이 JSON을 받아서 예쁜 UI 카드 형태로 화면에 그려주는 것이 현대적인 웹 개발의 기본 구조입니다!
