Chapter 5. 튼튼한 백엔드의 시작

Java 기초 (Full Course)

전 세계에서 가장 많이 쓰이는 백엔드 언어, 자바(Java)입니다. 엄격한 규칙과 '객체 지향'이라는 철학을 이해하면 거대한 시스템도 안정적으로 만들 수 있습니다.

1. 자바 프로그램의 뼈대

자바는 아주 작은 코드 하나를 짜더라도 반드시 클래스(Class)라는 큰 울타리와 main 메서드라는 출입문이 필요합니다.

// Main.java 파일 // 1. 클래스 (울타리) public class Main { // 2. main 메서드 (프로그램의 시작점) public static void main(String[] args) { // 화면에 출력하고 줄바꿈(ln) 해라! System.out.println("Hello, Java!"); } }
> Terminal - java Main
Hello, Java!
💡 파일 이름 규칙 자바에서는 public class의 이름과 파일의 이름이 반드시 100% 똑같아야 합니다. 클래스 이름이 Main이라면 파일 이름은 무조건 Main.java여야 합니다.

2. 엄격한 변수와 자료형

자바스크립트는 let 하나로 모든 변수를 만들었지만, 자바는 상자에 들어갈 물건의 종류(자료형)를 미리 정확하게 정해주어야 합니다.

대표적인 자료형 4가지

정수(int), 실수(double), 글자(String), 참/거짓(boolean)을 가장 많이 씁니다.

public class Main { public static void main(String[] args) { // int: 소수점이 없는 정수 int age = 20; // double: 소수점이 있는 실수 double pi = 3.14; // String: 문자열 (반드시 쌍따옴표) String name = "김자바"; // boolean: 참(true) 또는 거짓(false) boolean isStudent = true; System.out.println(name + "는 " + age + "살"); } }
> Terminal - java Main
김자바는 20살

3. 조건문과 반복문

프로그램의 흐름을 제어하는 논리 구조입니다. 이 부분은 파이썬이나 자바스크립트 등 다른 언어들과 거의 비슷합니다.

public class Main { public static void main(String[] args) { // 1. 조건문 (if - else) int score = 85; if (score >= 90) { System.out.println("A학점"); } else { System.out.println("B학점 이하"); } // 2. 반복문 (for) // 시작값; 조건식; 증감식 for (int i = 1; i <= 3; i++) { System.out.println(i + "번 반복"); } } }
> Terminal - java Main
B학점 이하
1번 반복
2번 반복
3번 반복

4. 객체 지향의 꽃 : 클래스와 객체 ★중요

자바를 배우는 진짜 이유입니다. 클래스(Class)는 '붕어빵 틀'이고, 그 틀로 찍어낸 실제 붕어빵들이 객체(Object/Instance)입니다.

나만의 데이터 타입 만들기

Car라는 설계도(클래스)를 만들면, 그 설계도로 수많은 자동차(객체)를 무한히 찍어낼 수 있습니다.

// 1. 자동차 설계도 (붕어빵 틀) class Car { String model; // 속성 (변수) int speed; // 동작 (메서드) void run() { System.out.println(model + "가 달립니다! 슝~"); } } public class Main { public static void main(String[] args) { // 2. 설계도로 실제 자동차 만들기 (new 키워드) Car myCar = new Car(); // 3. 속성 채워넣기 myCar.model = "테슬라"; myCar.speed = 120; // 4. 동작시키기 myCar.run(); } }
> Terminal - java Main
테슬라가 달립니다! 슝~
💡 new 키워드란? new는 "메모리(RAM)에 새로운 공간을 하나 만들어줘!"라는 뜻입니다. new Car()를 하는 순간 설계도가 비로소 실제 만질 수 있는 실체(객체)가 됩니다.

5. 메서드 (Methods)

클래스 안에 들어있는 '함수'를 자바에서는 '메서드'라고 부릅니다. 자바의 메서드는 결과물(반환값)의 자료형도 미리 정해두어야 합니다.

class Calculator { // int를 두 개(a, b) 받아서, int를 뱉어내는(return) 메서드 int add(int a, int b) { int result = a + b; return result; } // 반환할 게 없으면 void를 씁니다 void sayHello() { System.out.println("안녕하세요!"); } } public class Main { public static void main(String[] args) { Calculator calc = new Calculator(); int sum = calc.add(10, 20); System.out.println("두 수의 합: " + sum); calc.sayHello(); } }
> Terminal - java Main
두 수의 합: 30
안녕하세요!
