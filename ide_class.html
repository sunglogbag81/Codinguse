Chapter 1. 개발자의 무기 고르기

IDE (통합 개발 환경) 완벽 가이드

단종된 옛날 도구들은 전부 뺐습니다! 현재 실무에서 가장 많이 쓰이는 트렌디한 에디터와 IDE, 그리고 나에게 딱 맞는 무기를 찾아봅시다.

👑 1대장 & 새로운 도전자

어떤 언어를 배우든 가장 먼저 설치하게 될 범용 에디터들입니다.

🟦 VS Code (Visual Studio Code)

  • 추천 분야웹 프론트엔드(HTML/JS/React), Python, Node.js
  • 특징현재 전 세계 개발자 점유율 압도적 1위입니다. 가볍고 빠르며, 확장 프로그램(Extensions) 생태계가 미쳐서 원하는 기능을 무한히 붙일 수 있습니다. (무료)

⬜ Cursor (커서) ★요즘 대세

  • 추천 분야전 분야 (VS Code 대체재)
  • 특징VS Code를 똑같이 복사한 뒤, 그 안에 강력한 AI(ChatGPT 등)를 심어놓은 에디터입니다. "여기에 로그인 버튼 만들어줘"라고 치면 코드를 알아서 짜줍니다. 현재 씬에서 가장 뜨거운 도구입니다.

💎 JetBrains 제국 (프로들의 도구)

코드를 진짜로 '이해'하고 분석해주는 무거운 전문가용 도구들입니다. 대학교 이메일이 있다면 모두 무료로 쓸 수 있습니다.

🟥 IntelliJ IDEA & PyCharm

  • IntelliJ (Java/Spring)한국 백엔드 개발자들의 숨쉬는 공기와도 같습니다. 미친듯한 자동완성과 리팩토링 능력을 보여줍니다.
  • PyCharm (Python)파이썬 개발에 특화되어 있으며, 가상환경(venv) 세팅을 알아서 다 해줍니다.
  • 기타 형제들웹 개발용 WebStorm, C/C++용 CLion, C#용 Rider 등 언어별로 전용 에디터가 따로 존재합니다.

🟩 Android Studio (안드로이드 스튜디오)

  • 추천 분야안드로이드 앱 개발 (Kotlin, Java)
  • 특징구글이 JetBrains의 IntelliJ를 기반으로 만든 안드로이드 공식 개발 도구입니다. 스마트폰 화면을 띄워주는 에뮬레이터가 내장되어 있습니다.

🍎 대기업 전용 특수 무기

특정 생태계를 벗어나지 않는 거대한 플랫폼 전용 도구들입니다.

🟦 Xcode (엑스코드)

  • 추천 분야iOS(아이폰), Mac, Apple Watch 앱 개발
  • 특징애플 기기용 앱을 만들려면 선택권이 없습니다. 무조건 이거 써야 합니다. (맥북 필수)

🟪 Visual Studio (비주얼 스튜디오)

  • 추천 분야C, C++, C#, Unity(게임 개발), Windows 프로그램 개발
  • 특징디버깅(버그 잡기)의 신입니다. 프로그램이 매우 무겁지만(GB 단위), C언어를 배운다면 이만한 도구가 없습니다. (VS Code와 아예 다른 프로그램이니 헷갈리지 마세요!)

☁️ 무설치 웹 & 클라우드 에디터

프로그램 설치 없이, 브라우저만 켜면 바로 코딩할 수 있는 마법의 공간입니다. 아이패드나 크롬북에서도 개발이 가능합니다.

🟧 Google Colab (콜랩)

  • 추천 분야파이썬, 데이터 분석, 인공지능(AI) 모델 학습
  • 특징설치가 지옥 같은 파이썬 AI 환경을 구글이 다 세팅해 뒀습니다. 게다가 구글의 비싼 GPU 컴퓨터를 무료로 빌려 쓸 수 있습니다!

🔥 Replit / CodePen / Gitpod

  • Replit (리플릿)클라우드에 내 컴퓨터가 하나 생기는 느낌입니다. 파이썬, 노드 등 거의 모든 언어를 웹에서 5초 만에 실행할 수 있습니다.
  • CodePen (코드펜)HTML/CSS/JS 프론트엔드 연습장입니다. 화면이 어떻게 변하는지 실시간으로 보여줍니다.
  • GitHub Codespaces깃허브 저장소를 클릭 한 번에 웹버전 VS Code로 열어줍니다.

⚙️ 하드웨어 & 레거시(전통) 도구

🟦 Arduino IDE (아두이노)

  • 추천 분야IoT, 임베디드, 아두이노 보드 제어
  • 특징C/C++ 기반으로 센서나 모터를 제어하는 기계 코딩의 입문용 도구입니다.

🟧 Eclipse (이클립스)

  • 추천 분야과거 Java 프로젝트, 한국 전자정부프레임워크(SI 기업)
  • 특징과거 자바의 제왕이었습니다. 무겁고 투박하여 요즘은 거의 다 IntelliJ로 넘어갔지만, 공공기관이나 오래된 회사에서는 여전히 사용됩니다.

  • 📱 "아이폰 앱 만들 거다" 👉 Xcode
