C 프로그래밍 (Full Course)

현대 프로그래밍 언어들의 조상입니다. C언어를 배우면 컴퓨터가 메모리를 어떻게 다루고 작동하는지 그 진짜 원리를 깨우칠 수 있습니다.

1. 프로그램의 기본 구조

C언어는 요리를 하기 전 도구를 준비하듯, 표준 입출력 도구(stdio.h)를 먼저 가져오고 main 함수 안에서 모든 작업을 시작합니다.

// 1. 헤더 파일 포함 (Standard Input/Output) #include <stdio.h> // 2. 프로그램의 시작점 (main 함수) int main() { // 콘솔에 글자 출력. \n은 줄바꿈을 뜻합니다. printf("Hello, C Programming!\n"); // 운영체제에게 "에러 없이 정상 종료됨(0)"을 알림 return 0; }
> Terminal - ./main
Hello, C Programming!

2. 변수와 서식 지정자

C언어는 변수를 출력할 때 자바스크립트나 파이썬처럼 쉽게 섞어 쓸 수 없습니다. "%d 자리에 이 정수를 넣어줘"라는 방식으로 서식 지정자를 꼭 써야 합니다.

printf 와 scanf

%d는 정수(decimal), %f는 실수(float), %c는 문자 하나(character)를 의미합니다.

#include <stdio.h> int main() { int age = 20; float height = 175.5; char grade = 'A'; // %d, %f, %c 자리에 콤마 뒤의 변수들이 순서대로 들어감 printf("나이: %d살\n", age); printf("키: %.1fcm\n", height); // .1은 소수점 1자리까지 printf("등급: %c\n", grade); return 0; }
> Terminal - ./main
나이: 20살
키: 175.5cm
등급: A
💡 작은따옴표와 큰따옴표의 차이 C언어에서 'A'처럼 작은따옴표는 단 하나의 문자(char)를 의미하고, "Hello"처럼 큰따옴표는 문자열을 의미합니다. 절대 섞어서 쓰면 안 됩니다.

3. 제어문 (if와 for)

기본적인 조건문과 반복문은 다른 언어와 완전히 똑같습니다. C언어의 문법이 후대 언어들의 표준이 되었기 때문입니다.

#include <stdio.h> int main() { int score = 85; // 1. 조건문 if (score >= 90) { printf("합격\n"); } else { printf("불합격\n"); } // 2. 반복문 (구구단 2단 출력) for (int i = 1; i <= 3; i++) { printf("2 x %d = %d\n", i, 2 * i); } return 0; }
> Terminal - ./main
불합격
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6

4. 포인터 (Pointer) ★가장 중요

C언어의 하이라이트이자 최대 진입장벽입니다. 포인터는 데이터의 '값'이 아니라 데이터가 저장된 메모리의 '주소'를 보관하는 특수한 변수입니다.

주소(&)와 가리키는 값(*)

&를 붙이면 그 변수가 사는 집 주소(메모리 주소)를 알려주고, 포인터 변수 앞에 *를 붙이면 그 주소로 직접 찾아가 안의 값을 열어봅니다.

#include <stdio.h> int main() { int num = 10; // 포인터 변수 선언 (자료형 뒤에 *를 붙임) // &num 은 num 변수의 '메모리 주소'를 뜻함 int *ptr = # printf("num의 값: %d\n", num); printf("num의 주소: %p\n", ptr); // ptr 앞에 *를 붙이면 그 주소에 있는 '실제 값'을 의미함 printf("ptr이 가리키는 값: %d\n", *ptr); return 0; }
> Terminal - ./main
num의 값: 10
num의 주소: 0x7ffeefbff5a8
ptr이 가리키는 값: 10
💡 왜 포인터를 쓰나요? 만약 1GB짜리 데이터를 함수로 넘길 때, 데이터 전체를 복사하면 컴퓨터가 멈출 것입니다. 하지만 "이 데이터는 OOO번지 주소에 있어!"라고 포인터(주소표)만 넘기면 1초 만에 작업이 끝납니다.

5. 배열과 문자열

C언어에는 자바나 파이썬처럼 문자열(String)을 저장하는 전용 자료형이 없습니다. 대신 문자(char)들을 기차처럼 연결한 배열로 문자열을 표현합니다.

#include <stdio.h> int main() { // 1. 일반 숫자 배열 int scores[3] = {80, 90, 100}; printf("첫 번째 점수: %d\n", scores[0]); // 2. 문자 배열 (이것이 C언어의 문자열) // %s 서식 지정자를 사용하여 문자열을 출력합니다. char name[] = "Apple"; printf("단어: %s\n", name); printf("첫 글자: %c\n", name[0]); return 0; }
> Terminal - ./main
첫 번째 점수: 80
단어: Apple
첫 글자: A
