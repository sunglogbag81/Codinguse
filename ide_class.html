Chapter 1. 개발자의 무기 고르기

IDE (통합 개발 환경) 완벽 가이드

코드를 작성하고, 에러를 찾고, 실행까지 한 번에 해결해 주는 마법의 공구함! 수많은 에디터와 IDE 중 나에게 딱 맞는 무기를 찾아봅시다.

0. IDE가 뭔가요? (메모장 vs IDE)

코드는 사실 윈도우 메모장에 적어도 실행할 수 있습니다. 하지만 오타가 나도 안 알려주고, 실행하려면 까다로운 명령어를 직접 쳐야 합니다. IDE(Integrated Development Environment)는 코드 자동완성, 오타 감지, 원클릭 실행, Git 연동 등을 모두 제공하는 '풀옵션 작업대'입니다.

1. VS Code (Visual Studio Code)

현재 전 세계에서 가장 많이 쓰이는 압도적 1위의 코드 에디터입니다. 마이크로소프트에서 만들었으며, 가볍고 빠릅니다.

🟦 VS Code (Visual Studio Code)

  • 추천 분야웹 프론트엔드(HTML/CSS/JS/React), Python, Node.js
  • 장점프로그램 자체가 매우 가볍고, 무료입니다.
  • 특징'확장 프로그램(Extensions)' 생태계가 미쳤습니다. 테마부터 자동완성까지 원하는 기능을 레고 조립하듯 무한히 추가할 수 있습니다.
💡 헷갈림 주의! Visual Studio와 다릅니다! 이름이 비슷해서 뉴비들이 가장 많이 헷갈립니다. VS Code는 '가벼운 텍스트 에디터' 베이스이고, 밑에 설명할 Visual Studio는 '초대형 통합 개발 환경'입니다.

2. IntelliJ IDEA (인텔리제이)

Java 및 Spring Boot 백엔드 개발자들의 로망이자 필수품입니다. 체코의 JetBrains라는 회사에서 만들었습니다.

🟥 IntelliJ IDEA

  • 추천 분야Java, Spring Boot, Kotlin 백엔드 개발
  • 장점코드를 진짜 '이해'하고 추천해 주는 경이로운 자동완성(스마트 인텔리센스)과 강력한 코드 리팩토링 기능을 제공합니다.
  • 단점프로 버전(Ultimate)은 유료이며 꽤 비쌉니다. (단, 대학교 이메일이 있다면 학생 무료 라이선스를 받을 수 있습니다!)

3. Visual Studio (비주얼 스튜디오)

마이크로소프트의 근본이자, C언어 계열 개발자들의 영원한 파트너입니다.

🟪 Visual Studio

  • 추천 분야C, C++, C#, Unity(게임 개발), Windows 프로그램 개발
  • 장점C/C++ 디버깅(버그 잡기)에 있어서는 전 세계 원탑입니다. 설치만 하면 다른 세팅 없이 바로 코딩을 시작할 수 있습니다.
  • 단점프로그램이 엄청나게 무겁습니다. 설치 용량만 기가바이트(GB) 단위를 훌쩍 넘어가며, 가벼운 작업용으로는 과한 면이 있습니다. (무료 버전인 Community 에디션을 다운받으세요)

4. Eclipse (이클립스)

역사와 전통을 자랑하는 Java 전용 IDE입니다. 한국 IT 환경에서는 여전히 무시할 수 없는 점유율을 가집니다.

🟧 Eclipse

  • 추천 분야Java, 한국 SI 기업 및 공공기관 프로젝트 (전자정부표준프레임워크)
  • 장점완전한 오픈소스(무료)이며, 과거 수많은 기업의 표준 개발 툴로 자리 잡았습니다.
  • 단점디자인이 투박하고, 초기 환경 세팅이 복잡하며 무겁습니다. 최근에는 점차 IntelliJ로 넘어가는 추세입니다.

🎯 결론: 그래서 뭐 깔아야 해요?

  • 🌐 웹(HTML/JS/React)이나 파이썬을 배운다 👉 VS Code
  • ☕ Java나 Spring 백엔드를 배운다 👉 IntelliJ (학생 무료 필수!)
  • 🎮 C/C++ 이나 유니티(C#) 게임을 만든다 👉 Visual Studio
