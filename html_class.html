HTML 뼈대 세우기

어렵고 복잡한 연산은 필요 없습니다. 우리가 매일 보는 웹사이트의 구조를 텍스트로 블록 조립하듯 만들어보는 첫걸음입니다.

1. 태그(Tag)란 무엇인가?

HTML의 전부라고 해도 과언이 아닙니다. 브라우저에게 "이 텍스트는 제목이야", "이건 버튼이야"라고 알려주기 위해 이름표(태그)를 붙여주는 것이죠.

기본 문법: 열었으면 닫아라!

대부분의 태그는 한 쌍으로 움직입니다. <태그명>으로 열고, </태그명>으로 꼭 닫아주어야 합니다.

<h1>이것은 가장 큰 제목입니다</h1> <p>이것은 일반적인 문단(글)입니다. 여기에 내용을 적습니다.</p> <button>클릭하세요</button>

이것은 가장 큰 제목입니다

이것은 일반적인 문단(글)입니다. 여기에 내용을 적습니다.

2. HTML 문서의 기본 뼈대

사람에게 머리와 몸통이 있듯, HTML도 눈에 보이지 않는 설정을 담당하는 head와 실제 내용이 들어가는 body로 나뉩니다.

! + Enter (자동 완성)

VS Code에서 빈 파일에 !를 입력하고 Enter를 치면 이 뼈대가 자동으로 완성됩니다. 모든 코드는 <body> 안쪽에 작성하세요.

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <title>내 첫 번째 웹사이트</title> </head> <body> <!-- 우리가 브라우저에서 보는 모든 것은 여기에 적습니다 --> </body> </html>

3. 링크와 이미지 (속성)

태그만으로는 정보가 부족할 때가 있습니다. "어디로 이동할지", "무슨 이미지를 띄울지" 등 추가 정보를 태그 안에 적어주는 것을 속성(Attribute)이라고 합니다.

a 태그 (링크 연결)와 img 태그

a 태그는 href 속성을, img 태그는 src 속성을 사용합니다. 참고로 img는 닫는 태그가 없는 대표적인 태그입니다.

<!-- 구글로 이동하는 링크 --> <a href="https://google.com">구글 바로가기</a> <!-- 이미지 띄우기 --> <img src="https://via.placeholder.com/150" alt="임시 사진">
구글 바로가기
💡 alt 속성이란? 이미지 엑박이 떴을 때 대신 보여줄 글자입니다. 시각장애인용 화면 판독기가 읽어주는 중요한 정보이므로 꼭 써주는 습관을 들이세요!

4. 구역 나누기 (div와 span)

요소들을 예쁘게 묶고 배치하기 위해 사용하는 가장 흔한 상자(박스)입니다. 다음 시간에 배울 CSS(디자인)를 적용하기 위해 필수적입니다.

블록(Block)과 인라인(Inline)

div는 가로 전체를 차지해 자동으로 줄바꿈이 되고, span은 내용물 크기만큼만 차지해 한 줄에 나란히 배치됩니다.

<div style="background: yellow;">나는 div입니다</div> <div style="background: orange;">한 줄을 다 차지하죠</div>
<span style="background: lightblue;">나는</span> <span style="background: lightgreen;">span입니다</span>
나는 div입니다
한 줄을 다 차지하죠
나는 span입니다
