HTML 뼈대 세우기 (Full Course)

가장 기초적인 태그부터 폼(Form) 구성, 시맨틱 웹까지. 이 페이지 하나로 HTML의 모든 핵심 개념을 마스터할 수 있습니다.

1. 태그(Tag)와 텍스트 포맷팅

HTML은 브라우저에게 "이건 제목이야", "이건 강조할 글자야"라고 이름표(태그)를 붙여주는 언어입니다. <태그명>으로 열고 </태그명>으로 닫습니다.

제목, 문단, 그리고 텍스트 꾸미기

단순한 글자 출력을 넘어, 글자의 굵기나 기울기, 형광펜 효과 등을 줄 수 있습니다.

<h1>가장 큰 제목</h1> <h3>중간 제목</h3> <p> 일반 문단입니다. <br> <strong>중요한 글자(굵게)</strong><br> <em>강조하는 글자(기울임)</em><br> <mark>형광펜 효과</mark><br> <del>취소선</del> </p>

2. 문서의 뼈대 구성

모든 HTML 문서는 눈에 보이지 않는 설정을 담당하는 <head>와 실제 화면에 그려지는 <body>로 나뉩니다. VS Code에서 ! + Enter를 누르면 자동 완성됩니다.

<!-- 문서가 HTML5임을 명시 --> <!DOCTYPE html> <html lang="ko"> <head> <!-- 한글이 깨지지 않도록 인코딩 설정 --> <meta charset="UTF-8"> <!-- 모바일 화면 크기 맞춤 설정 --> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 브라우저 탭에 표시될 제목 --> <title>내 웹사이트</title> </head> <body> <!-- 웹 브라우저 화면에 보일 모든 내용은 여기에 --> <h1>안녕하세요!</h1> </body> </html>

3. 속성(Attribute)과 멀티미디어

"어디로 이동할지", "어떤 파일을 재생할지" 등 부가적인 명령을 태그 안에 적어주는 것을 속성이라고 합니다.

링크, 이미지, 비디오, 오디오

멀티미디어 태그들은 src(Source, 파일 경로) 속성을 필수로 가집니다.

<!-- 1. 링크 (target="_blank"는 새 탭 열기) --> <a href="https://google.com" target="_blank">구글</a> <!-- 2. 이미지 (alt는 시각장애인용 대체 텍스트) --> <img src="dog.jpg" alt="귀여운 강아지" width="100"> <!-- 3. 비디오 (controls는 재생 버튼 표시) --> <video src="movie.mp4" controls width="200"></video> <!-- 4. 오디오 --> <audio src="music.mp3" controls></audio>

4. 목록(List)과 표(Table)

데이터를 깔끔하게 나열할 때 사용합니다. 표(Table)는 과거 웹 레이아웃에 쓰였지만, 현재는 순수하게 표 형태의 데이터를 보여줄 때만 사용합니다.

<!-- 순서 없는 목록 (Unordered List) --> <ul> <li>HTML</li> <li>CSS</li> </ul> <!-- 순서 있는 목록 (Ordered List) --> <ol> <li>기획하기</li> <li>개발하기</li> </ol> <!-- 표 만들기 --> <table> <thead> <!-- 표 제목 영역 --> <tr> <!-- Table Row (가로줄) --> <th>과목</th> <!-- Table Header (제목 셀) --> <th>점수</th> </tr> </thead> <tbody> <!-- 표 본문 영역 --> <tr> <td>수학</td> <!-- Table Data (일반 셀) --> <td>100</td> </tr> </tbody> </table>

5. 사용자 입력 받기 (Form)

로그인, 회원가입, 댓글 쓰기 등 사용자가 서버로 데이터를 보낼 때 <form> 태그와 다양한 입력(Input) 태그들을 사용합니다.

<form> <!-- 짧은 글자 입력 --> <input type="text" placeholder="아이디"><br> <input type="password" placeholder="비번"><br> <!-- 여러 개 중 하나 선택 (라디오 버튼) --> <input type="radio" name="gender">남 <input type="radio" name="gender">여<br> <!-- 다중 선택 (체크박스) --> <input type="checkbox"> 약관 동의<br> <!-- 드롭다운 선택 --> <select> <option>SKT</option> <option>KT</option> </select><br> <!-- 긴 글 입력창 --> <textarea rows="3" placeholder="내용"></textarea><br> <!-- 제출 버튼 --> <button type="submit">가입하기</button> </form>

6. 공간 분할과 시맨틱(Semantic) 태그

웹페이지를 디자인하려면 블록 단위로 구역을 묶어야 합니다. 이때 가장 많이 쓰는 것이 div와 span입니다.

id 와 class 속성

수많은 div를 구분하기 위해 별명을 붙여줍니다. 이는 CSS와 JavaScript에서 조작할 때 핵심 역할을 합니다.

<!-- class: 중복 가능한 별명 (디자인용) --> <div class="box card-item">1번 카드</div> <div class="box card-item">2번 카드</div> <!-- id: 문서 내 유일한 이름 (주요 식별용) --> <div id="main-header">메인 상단</div>

시맨틱(의미론적) 태그

과거에는 모든 구역을 <div id="header">처럼 나눴지만, HTML5부터는 이름 자체에 의미가 담긴 태그를 사용합니다. 이는 구글 검색엔진(SEO) 최적화에 큰 도움을 줍니다.

<header> 웹페이지의 맨 위 헤더 영역 </header> <nav> 메뉴(내비게이션) 영역 </nav> <main> 본문 핵심 영역 </main> <section> 본문 내부의 구분된 구역 </section> <article> 독립적인 글(게시물) 영역 </article> <footer> 맨 아래 사업자정보 등 푸터 영역 </footer>
💡 알아두기 시맨틱 태그들은 기능적으로는 div와 100% 동일합니다. (화면 전체 가로를 차지하는 블록 요소) 단지 컴퓨터가 "아, 여기가 메뉴구나"라고 이해할 수 있게 도와줄 뿐입니다.
