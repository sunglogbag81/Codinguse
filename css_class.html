CSS 스타일링 (Full Course)

HTML로 세운 뼈대에 색을 칠하고, 간격을 맞추고, 예쁘게 배치하는 마법. CSS의 핵심 원리인 박스 모델과 Flexbox를 완벽히 이해해 봅시다.

1. 선택자 (Selectors)

CSS의 기본 공식은 누구에게 { 어떤스타일을 : 줄것인가; } 입니다. 수많은 태그 중에서 내가 꾸미고 싶은 녀석만 콕 집어내는 방법이 바로 선택자입니다.

태그, 클래스(Class), 아이디(ID)

클래스는 .(점)으로, 아이디는 #(샵)으로 선택합니다. 실무에서는 재사용이 가능한 클래스(Class)를 가장 많이 사용합니다.

/* 1. 태그 선택자 (모든 p 태그 변경) */ p { color: gray; } /* 2. 클래스 선택자 (별명) - 가장 많이 씀 */ .highlight { color: blue; font-weight: bold; } /* 3. 아이디 선택자 (문서 내 유일) */ #title { color: red; }

2. 텍스트와 색상 다루기

글자의 크기, 굵기, 정렬, 그리고 글자색과 배경색을 지정하는 가장 기본적인 속성들입니다.

.box { /* 색상 (Hex 코드 방식) */ color: #ffffff; /* 글자색: 흰색 */ background-color: #3b82f6; /* 배경색: 파란색 */ /* 폰트 스타일 */ font-size: 20px; /* 글자 크기 */ font-weight: bold; /* 글자 굵기 (normal, bold 등) */ /* 정렬 및 기타 */ text-align: center; /* 가운데 정렬 */ line-height: 1.5; /* 줄 간격 */ }

3. 박스 모델 (Box Model) ★중요

HTML의 모든 요소는 네모난 '박스'입니다. 안쪽 여백(Padding)과 바깥쪽 여백(Margin)을 확실히 구분해야 레이아웃이 깨지지 않습니다.

Padding, Border, Margin

내용물을 감싸는 안쪽 여백이 Padding, 테두리가 Border, 다른 요소들을 밀어내는 바깥 여백이 Margin입니다.

.my-box { width: 100%; /* 테두리: 굵기, 종류, 색상 */ border: 4px solid #3b82f6; /* 테두리 안쪽 여백 (내용물이 숨쉬는 공간) */ padding: 20px; /* 테두리 바깥쪽 여백 (남들과의 거리) */ margin-bottom: 15px; }
(이전 요소)
Padding 20px (안쪽 여백) Content (내용물)
↑ Margin 15px (바깥 여백) ↓
(다음 요소)
💡 margin: 0 auto; 란? 요소의 가로 너비(width)가 정해져 있을 때, margin: 0 auto;를 주면 좌우 마진을 똑같이 나누어 가져 요소가 화면 정가운데로 정렬됩니다.

4. 배치의 마법 : Flexbox

현대 웹 디자인에서 절대 빠질 수 없는 레이아웃 기술입니다. 요소들을 가로로 나란히 세우거나 화면 정중앙에 배치할 때 매우 강력합니다.

부모에게 명령하면 자식이 움직입니다

감싸고 있는 부모 상자에 display: flex;를 주면, 자식 요소들이 기본적으로 가로(왼쪽에서 오른쪽)로 정렬됩니다.

.container { /* 플렉스박스 선언! */ display: flex; /* 가로축 정렬 방법 */ /* space-between: 양끝으로 벌리기 */ /* center: 가운데 모으기 */ justify-content: space-between; /* 세로축 정렬 방법 */ align-items: center; /* 자식들 사이의 간격 */ gap: 10px; }

5. 위치 지정 : Position

요소를 정상적인 흐름에서 벗어나게 하여, 원하는 좌표에 강제로 띄우거나 고정시킬 때 사용합니다. (예: 둥둥 떠다니는 헤더, 화면 우측 하단의 위로가기 버튼)

.parent { /* 자식의 기준점이 됨 (필수) */ position: relative; } .child { /* 부모를 기준으로 공중에 뜸 */ position: absolute; top: 10px; /* 위에서 10px */ right: 10px; /* 오른쪽에서 10px */ } .fixed-btn { /* 화면(브라우저)을 기준으로 고정됨 */ position: fixed; bottom: 20px; }

6. 반응형 웹 (Media Query)

PC와 모바일 화면 크기에 따라 디자인을 다르게 보여주는 기술입니다. 현재 여러분이 보고 계신 이 페이지도 미디어 쿼리로 만들어졌습니다.

/* 기본 스타일 (PC 기준) */ .box { width: 500px; font-size: 20px; } /* 가로 크기가 768px 이하(모바일)일 때만 작동하는 코드 */ @media (max-width: 768px) { .box { width: 100%; /* 화면에 꽉 차게 */ font-size: 14px; /* 글씨는 작게 */ } }
💡 개발자 도구 (F12) 브라우저에서 F12를 눌러 개발자 도구를 켜고 모바일 기기 아이콘을 클릭해 보세요. 화면 크기를 줄였다 늘였다 하면서 반응형이 어떻게 동작하는지 실시간으로 확인할 수 있습니다.
