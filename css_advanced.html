CSS 심화 (Advanced)

Flexbox를 넘어선 2차원 Grid 레이아웃, 생동감을 불어넣는 애니메이션, 그리고 HTML을 수정하지 않고 요소를 추가하는 가상 요소의 마법을 배웁니다.

1. CSS Grid (2차원 레이아웃의 끝판왕)

Flexbox가 가로 혹은 세로 한 방향(1차원)으로만 정렬한다면, Grid는 바둑판처럼 가로와 세로(2차원)를 동시에 제어하는 강력한 레이아웃 시스템입니다. 갤러리나 복잡한 대시보드를 만들 때 필수입니다.

.container { display: grid; /* 1fr(비율) 크기로 3칸을 만들어라 */ grid-template-columns: repeat(3, 1fr); gap: 10px; } .item-wide { /* 이 아이템은 가로로 2칸을 차지해라 */ grid-column: span 2; }

💡 1fr (Fraction) 단위란? Grid에서 쓸 수 있는 비율 단위입니다. `1fr 2fr 1fr`로 설정하면 화면을 1:2:1 비율로 쪼개줍니다. 남은 공간을 알아서 계산해주므로 %보다 훨씬 편리합니다.

2. 가상 클래스와 가상 요소 (Pseudo)

HTML을 전혀 건드리지 않고, CSS만으로 마우스를 올렸을 때(`:hover`), 특정 번째 자식(`:nth-child`), 심지어 새로운 요소를 창조(`::before`, `::after`)할 수 있습니다.

::after 를 이용한 툴팁 만들기

가상 요소를 사용하려면 content: "" 속성이 필수입니다. 이를 활용해 자바스크립트 없이 툴팁을 띄워봅시다.

.tooltip-btn { position: relative; } /* 버튼 뒤(after)에 가상의 요소를 하나 생성함 */ .tooltip-btn::after { content: "짜잔! 가상 요소입니다"; position: absolute; opacity: 0; /* 평소엔 투명하게 숨김 */ transition: 0.3s; } /* 버튼에 마우스가 올라가면(hover) 가상요소 보이기 */ .tooltip-btn:hover::after { opacity: 1; }

3. 모던 CSS 함수 (calc, clamp)

요즘은 미디어 쿼리(`@media`) 없이도 폰트 크기나 여백이 화면 크기에 맞춰 물 흐르듯 변하는 Fluid Typography(유동적 타이포그래피)를 많이 사용합니다.

.box { /* calc: CSS 안에서 사칙연산 계산기 돌리기 */ /* 전체 너비(100%)에서 양옆 여백 50px씩 뺀 크기 */ width: calc(100% - 100px); /* clamp: 최솟값, 권장값, 최댓값 설정 */ /* 화면이 작아도 16px 밑으론 안 내려가고, 커져도 32px 위론 안 커짐. 그 사이는 화면 비율(vw)에 맞춰 스무스하게 커짐 */ font-size: clamp(16px, 4vw, 32px); }

4. 트랜지션과 애니메이션

transition은 A 상태에서 B 상태로 부드럽게 넘어갈 때 쓰고, @keyframes animation은 내가 원하는 다채로운 프레임 단위의 움직임을 무한히 반복할 때 사용합니다.

/* 1. 키프레임 (애니메이션 대본 작성) */ @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { /* 위로 30px 튕겨 오름 */ transform: translateY(-30px); } } .ball { /* 2. 작성한 대본을 1초 동안, 무한히, 부드럽게 실행 */ animation: bounce 1s infinite ease-in-out; }
💡 성능 최적화 팁 애니메이션을 줄 때 `top`, `left`, `margin` 등을 변경하면 브라우저가 화면을 매번 다시 그려야 해서 렉이 걸립니다. 반드시 `transform`과 `opacity`만 이용해 애니메이션을 만들어야 60fps로 부드럽게 동작합니다.

5. 3D Transform (카드 뒤집기)

CSS의 transform 속성 중 rotateY, perspective(원근감)를 조합하면 완벽한 3D 입체 효과를 낼 수 있습니다.

.card-container { /* 3D 원근감 부여 (값이 작을수록 왜곡이 심함) */ perspective: 1000px; } .card-inner { transition: transform 0.6s; /* 자식 요소들도 3D 공간을 유지하도록 설정 */ transform-style: preserve-3d; } /* 카드에 마우스를 올리면 Y축으로 180도 회전 */ .card-container:hover .card-inner { transform: rotateY(180deg); } /* 카드의 뒷면은 평소에 안 보이게 처리 */ .card-back { backface-visibility: hidden; transform: rotateY(180deg); }

6. 트렌디한 효과: Glassmorphism

요즘 애플(Apple)이나 최신 웹사이트에서 가장 많이 쓰는 '반투명한 유리 질감' 효과입니다. backdrop-filter 하나면 끝납니다.

.glass-card { /* 1. 배경을 반투명한 흰색으로 */ background: rgba(255, 255, 255, 0.2); /* 2. 요소의 '배경(뒤쪽)'을 블러 처리 (핵심) */ backdrop-filter: blur(10px); /* 3. 살짝 흰색 테두리를 주면 유리 느낌 극대화 */ border: 1px solid rgba(255, 255, 255, 0.3); }
