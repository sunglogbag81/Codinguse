JavaScript 기초 (Full Course)

단순한 문서를 넘어 '프로그램'으로 진화하는 시간입니다. 변수, 함수 같은 프로그래밍 기초부터 HTML을 직접 조작하는 DOM 제어까지 한 번에 알아봅시다.

1. 변수 (Variables)

데이터를 담아두는 상자입니다. 과거에는 var를 썼지만, 요즘은 let(변하는 값)과 const(변하지 않는 값)만 사용합니다.

// 1. let: 나중에 값을 바꿀 수 있음 let age = 20; age = 21; // 에러 없음 // 2. const: 한 번 넣으면 바꿀 수 없음 (상수) const name = "홍길동"; // name = "김코딩"; // ❌ 에러 발생! // 콘솔창에 출력해보기 console.log(name + "은 " + age + "살");
> Console Output
홍길동은 21살
💡 실무 꿀팁 일단 무조건 const로 선언하세요. 나중에 코드를 짜다가 "아, 이 값은 변경되어야 하네?" 싶을 때만 let으로 바꾸는 것이 버그를 줄이는 지름길입니다.

2. 함수 (Functions)

자주 쓰는 코드 묶음을 캡슐에 담아놓고 자판기 버튼 누르듯 필요할 때마다 꺼내 쓰는 기능입니다.

기본 함수와 화살표 함수

함수는 재료(매개변수)를 받아서 결과(반환값)를 내놓습니다. 최근에는 화살표 함수(Arrow Function)를 더 많이 씁니다.

// 1. 기본 함수 선언 방식 function add(a, b) { return a + b; } // 2. 화살표 함수 방식 (최신 문법) const multiply = (a, b) => { return a * b; }; // 함수 사용(호출)하기 let result1 = add(5, 3); // 결과: 8 let result2 = multiply(4, 2); // 결과: 8

3. DOM 제어 (HTML 조작하기) ★중요

자바스크립트가 웹에서 하는 가장 중요한 역할입니다. Document Object Model의 약자로, JS를 통해 HTML 태그들을 선택하고 내용을 바꾸거나 스타일을 입힐 수 있습니다.

요소 선택하고 텍스트 바꾸기

document.querySelector('선택자')를 사용하면 CSS에서 하던 방식 그대로 HTML 요소를 콕 집어올 수 있습니다.

<!-- HTML --> <h1 id="title">원래 제목</h1> /* JS */ // 1. HTML 요소 가져오기 const titleElement = document.querySelector('#title'); // 2. 안쪽 텍스트 바꾸기 titleElement.innerText = "JS가 바꾼 제목!"; // 3. CSS 스타일 바꾸기 titleElement.style.color = "blue";

4. 이벤트 (Events)

사용자가 버튼을 '클릭'하거나, 키보드를 '입력'하는 등의 모든 행동을 '이벤트'라고 합니다. 이 이벤트가 발생했을 때 특정 함수가 실행되게 만들 수 있습니다.

addEventListener

"이 버튼이 클릭되는지 귀를 기울여라(Listen)!"라는 뜻입니다. 인터랙티브한 웹은 모두 이 원리로 돌아갑니다.

<!-- HTML --> <button id="myBtn">여기를 눌러보세요</button> <p id="msg"></p> /* JS */ const btn = document.querySelector('#myBtn'); const msg = document.querySelector('#msg'); // 버튼에 'click' 이벤트 달아주기 btn.addEventListener('click', () => { msg.innerText = "버튼이 클릭되었습니다! 🎉"; btn.style.backgroundColor = "green"; });

5. 실전 종합: 로그인 흉내내기

입력창(Input)에서 값을 읽어와서 조건문(if)으로 검사하고, 결과를 화면(DOM)에 띄워보는 종합 예제입니다.

const input = document.querySelector('#userId'); const loginBtn = document.querySelector('#loginBtn'); const result = document.querySelector('#result'); loginBtn.addEventListener('click', () => { // input 태그에 적힌 글자는 .value 로 가져옵니다 let idValue = input.value; if (idValue === "admin") { result.innerText = "관리자님 환영합니다."; result.style.color = "blue"; } else if (idValue === "") { result.innerText = "아이디를 입력해주세요."; result.style.color = "red"; } else { result.innerText = idValue + "님 로그인 성공!"; result.style.color = "green"; } });

⚠️ 많이 하는 실수 일반적인 태그의 글씨는 .innerText로 바꾸고 가져오지만, <input> 태그 안에 사용자가 타이핑한 값은 무조건 .value로 가져와야 합니다!
