1. 편해진 입출력 (iostream)

C언어의 printf와 scanf처럼 복잡한 %d, %f 서식 지정자를 외울 필요가 없습니다. 화살표 모양의 스트림(<<, >>) 기호를 사용하여 물 흐르듯 데이터를 넣고 뺍니다.

#include <iostream> using namespace std; // std:: 를 생략하게 해줌 int main() { int age; // cout (Console Out): 화면으로 데이터를 보냄 (출력) cout << "나이를 입력하세요: "; // cin (Console In): 입력을 받아서 변수에 넣음 cin >> age; // endl 은 엔터(줄바꿈)를 의미함 cout << "당신의 나이는 " << age << "살 입니다." << endl; return 0; }
> Terminal - ./main
나이를 입력하세요: 20
당신의 나이는 20살 입니다.

2. 강력해진 도구: String과 Vector

C언어에서는 문자열이나 크기가 변하는 배열을 다루는 것이 지옥이었습니다. C++에서는 STL(표준 템플릿 라이브러리) 덕분에 파이썬처럼 쉽게 데이터를 다룰 수 있습니다.

문자열(String)과 동적 배열(Vector)

string은 문자열을 무한히 더할 수 있고, vector는 배열이 꽉 차면 스스로 크기를 늘리는 마법의 배열입니다.

#include <iostream> #include <string> #include <vector> using namespace std; int main() { // 1. String: C언어의 char[] 보다 훨씬 편합니다. string name = "Hello"; name = name + " C++!"; // 덧셈으로 문자열 붙이기 가능 // 2. Vector: 크기가 자유자재로 변하는 배열 vector<int> scores = {80, 90}; // push_back: 배열의 맨 끝에 데이터 추가 scores.push_back(100); cout << name << endl; cout << "총 과목 수: " << scores.size() << endl; return 0; }
> Terminal - ./main
Hello C++!
총 과목 수: 3

3. 포인터 대신 레퍼런스(Reference)

C언어에서 메모리 주소를 넘길 때 썼던 복잡한 포인터(`*`와 `&`의 늪)를 대체하기 위해, C++에서는 참조자(Reference, `&`)라는 개념을 도입했습니다. 변수의 '별명'을 지어주는 것과 같습니다.

#include <iostream> using namespace std; // 레퍼런스(&)를 사용하면 원본 데이터를 직접 수정할 수 있음 // C언어처럼 포인터(*)를 쓰지 않아도 됨! void swap(int& a, int& b) { int temp = a; a = b; b = temp; } int main() { int x = 10, y = 20; swap(x, y); // 원본 변수를 그대로 넘김 cout << "x: " << x << ", y: " << y << endl; return 0; }
> Terminal - ./main
x: 20, y: 10
💡 왜 포인터를 안 썼나요? 만약 함수에서 `int a` 로 그냥 값을 받으면 원본이 복사되어 넘어옵니다(값에 의한 호출). 하지만 `int& a` 로 참조자를 사용하면, 복사하지 않고 원본 메모리에 직접 연결되므로 메모리도 아끼고 원본 값도 수정할 수 있습니다.

4. 객체지향 프로그래밍 (Class) ★중요

C++의 존재 이유입니다. C언어의 구조체(struct)에 '함수'를 함께 묶고, 데이터를 숨길 수 있는(private) 캡슐화 기능을 더해 완벽한 객체지향 언어로 진화했습니다.

클래스(설계도)와 접근 제어자

private는 클래스 외부에서 접근할 수 없게 막고, public은 외부에서 접근을 허용합니다.

#include <iostream> using namespace std; // 1. 플레이어 설계도(Class) 생성 class Player { private: // 체력은 외부에서 맘대로 못 바꾸게 숨김 int hp; public: // 생성자 (객체가 태어날 때 초기화) Player(int initialHp) { hp = initialHp; } // 외부에서 호출할 수 있는 함수 void takeDamage(int damage) { hp = hp - damage; cout << "앗! 남은 체력: " << hp << endl; } }; int main() { // 2. 설계도를 바탕으로 객체(인스턴스) 생성 Player hero(100); // hero.hp = 0; // ❌ private 이라 불가능! (핵 방지) // 3. public 메서드를 통해 상호작용 hero.takeDamage(30); return 0; }
> Terminal - ./main
앗! 남은 체력: 70
